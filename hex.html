Text-Hex Converter
Hexadecimal(base-16) system is an alternative of decimal(base-10) system which uses 16 symbols to represent numbers instead of 10.
While decimal system uses the symbols '0'-'9' to represent values from 0 to 9, hexadecimal system uses the symbols '0'-'9' to represent values from 0 to 9, and 'a'-'f' to represent values from 10 to 15.
Example - The letter 'L' of the alphabet is '4c' in hex.
Usage
To convert text or sentence to hex, type in what you want to convert in the input text box and click 'Text to Hex'.
To convert hex to text, type in the hex(space seperated between each pair) in the input text box and click 'Hex to Text.
